# Classify
## Как запустить приложение локально?
1. Скачайте и установите Python 3. Убедитесь, что Python 3 установлен: в командном интерпретаторе (cmd, bash или любом другом) введите `python --version`, вы должны увидеть версию Python.

2. Добавьте в файл input.txt, который располагается в папке приложения, (или любой другой файл) изображения в формате Base64. Изображения должны включать `data:[<тип данных>][;base64],<данные>`.

3. Запустите локальный сервер и приложение, кликнув по bat-файлу.
